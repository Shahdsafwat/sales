# 📊 Sales Report Dashboard

An interactive Power BI dashboard analyzing sales performance across US states, product categories, and months.

## 📈 Key Metrics
- **Number of Orders:** 178.41K
- **Total Sales:** $34.46M
- **Average Order Value:** $185.6
- **Best Month:** December ($4.61M)

## 📌 Dashboard Highlights

- **Sum of Total Price by State** — Geographic map view across US states (California, Georgia, Massachusetts, New York, Oregon, Texas, Washington)
- **Total Sales by Month and Category** — Clustered bar chart across all months for 5 categories (Headphones, Laptop, Machine, Monitor, Phone)
- **Monthly Sales Trend** — Line chart showing sales growth peaking in December
- **Total Price and Total Orders by Category** — Combined bar & line chart (Laptop, Phone, Monitor, Headphones, Machine, Cable, Batteries)

## 🔍 Filters & Slicers
- State filter buttons
- Month slicer
- Category slicer
- Clear all slicers button

## 🗂️ Dataset Fields
- Order ID, Order Date, Product, Category
- Price Each, Quantity Ordered, Total Price, State

## 🛠️ Tools Used
- Power BI Desktop
- Power Query
